# Project Title: Australian Disability Employment Analysis

#### Technical Stack: SQL (MySQL), Power BI (DAX & Power Query), GitHub

#### Project Overview:
Developed an end-to-end data pipeline and interactive dashboard to identify employment barriers and workforce participation gaps for Australians with disabilities. The project transformed complex ABS-style datasets into actionable insights to support evidence-based policy and diversity decision-making. 

#### Key Contributions:
--- Data Engineering (SQL): Engineered a scalable ETL process in MySQL to unpivot and normalise three complex relational tables (Living Arrangements, Labour Force Status, and Industry Occupations), ensuring data was "dashboard-ready". 
--- Advanced Analytics (Power BI/DAX): Designed a high-contrast, color-blind-accessible dashboard featuring custom DAX measures for Unemployment Rates and Workforce Participation Rates, specifically benchmarking different disability cohorts.
--- Insight Generation: Identified critical correlations between Supported Living Arrangements and Employment using line plot analysis and "Top-N" industry filtering.
--- Deployment (GitHub): Managed version control via GitHub, documenting the SQL transformation logic and Power BI Project (.pbip) files to ensure transparency and reproducibility.
--- Professional Impact: Optimised data processing workflows by replacing manual Excel transformations with automated SQL scripts, reducing report refresh time and improving data integrity for Australia-focused reporting.
